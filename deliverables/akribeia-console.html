The quant engine

Every screen builds on this. ~1,289 US stocks + ~70 ETFs scored daily.

Five pillars → composite → rating

ValuationGrowthProfitabilityMomentumEPS Revisions

Each graded A+→F (sector-relative), rolled into a composite. Top names → Strong Buy+ / Strong Buy / Buy; rest → Hold / Sell / Strong Sell.

Three price anchors

Fair Value — intrinsic value + premium/discount + verdict.

Quant Buy Point — the model's entry, with BP distance + signal.

FCF quality — free-cash-flow yield after expensing stock comp.

Badges — S&P / NDX index membership (SPY ∪ Wikipedia).
